# infinity_tool_scripts
浏览器插件 —— Infinity Pro 专用增强脚本，一些简单的常用功能



## 脚本功能

| 脚本名称                   | 功能                          |
| -------------------------- | ----------------------------- |
| infinity_append_html       | pro版本数据生成旧版可导入文件 |
| infinity_convert_oldbackup | 旧版文件数据导入新版pro       |
| infinity_export_excel      | pro版本网站导出excel文件      |
